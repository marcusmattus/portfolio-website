## Getting Started

1. **Add Your Images**
   - Replace `images/profile.jpg` with your profile picture
   - Add project screenshots (project1.jpg, project2.jpg, project3.jpg)
   - Add an about section image (about.jpg)

2. **Customize Content**
   - Update the name "Marcus Mattus" throughout the HTML
   - Modify the hero section description
   - Update the about section with your story
   - Replace project information with your actual projects
   - Update contact information
   - Add your social media links

3. **Customize Colors**
   - Edit CSS variables in `:root` section of `style.css`
   - Change `--primary-color`, `--secondary-color`, etc.

4. **Run the Website**
   - Open `index.html` in a web browser
   - Or use a local server: `python -m http.server 8000`

## Customization

### Colors
Edit the CSS variables at the top of `style.css`:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #f093fb;
    /* ... */
}
```

### Content
- Update personal information in `index.html`
- Modify project details and links
- Add your actual skills and technologies
- Update contact information
